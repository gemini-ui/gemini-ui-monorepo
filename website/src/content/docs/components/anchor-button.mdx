The `AnchorButton` component is used to navigate to a different page or section of the same page.

```astro live
---
import { AnchorButton } from "@gemini-ui-astro/components";
---

<AnchorButton href="https://www.google.com" target="_blank">
  Google
</AnchorButton>
```
